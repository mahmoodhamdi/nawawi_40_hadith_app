## Full description (max 4000 chars)

**Arba'in An-Nawawi — Dengan Suara Syaikh Ahmad Al-Nafees**
**Sedekah jariyah open source — Tanpa iklan, tanpa pelacakan, tanpa internet**

Aplikasi Islami gratis yang menyajikan 42 Hadits yang dihimpun oleh Imam An-Nawawi rahimahullah, lengkap dengan tilawah jelas oleh Syaikh Ahmad Al-Nafees dan penjelasan sederhana untuk setiap hadits.

### Mengapa aplikasi ini?

✓ **Sepenuhnya offline** — Semua teks dan audio sudah disertakan
✓ **Tanpa iklan** — Tidak ada iklan, tidak ada tautan komersial
✓ **Tanpa pelacakan** — Tidak ada analytics, privasi terjaga 100%
✓ **Open source** — Kode tersedia di GitHub
✓ **Sedekah jariyah** — Setiap bacaan dan unduhan menambah timbangan amal

### Fitur

📖 **Sanad terverifikasi** untuk setiap hadits (rawi, sumber, tautan sunnah.com)
🔊 **Audio tilawah** oleh Syaikh Ahmad Al-Nafees
📚 **Penjelasan sederhana** untuk setiap hadits
🔍 **Pencarian cerdas** berdasarkan teks atau nomor hadits
⭐ **Favorit** untuk menandai hadits penting
📊 **Lanjutkan baca** dari hadits terakhir
🎨 **Empat tema** (Terang, Gelap, Biru, Ungu)
🌍 **Dua bahasa**: Arab dan Inggris (Bahasa Indonesia menyusul)
🔔 **Pengingat harian** sesuai waktu yang Anda pilih
🎯 **Mode baca fokus** untuk perenungan dan hafalan
🖼️ **Bagikan hadits sebagai gambar** ke WhatsApp/Twitter/Instagram

### Niat

Aplikasi ini adalah sedekah jariyah untuk setiap Muslim. Tujuan kami adalah menyampaikan ilmu Imam An-Nawawi rahimahullah kepada sebanyak mungkin Muslim di seluruh dunia. Ikut serta dalam pahala dengan membagikan aplikasi ini kepada minimal 3 teman.

### Privasi

- Tidak mengumpulkan data pribadi
- Tidak ada SDK pelacakan
- Sepenuhnya offline
- Izin yang dibutuhkan: Notifikasi (untuk pengingat harian)

Semoga Allah menjadikan ini sebagai sedekah jariyah bagi yang membuat dan turut menyebarkannya.
